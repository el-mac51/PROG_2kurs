## В папке представлены результаты выполнения лабораторной работы номер 6.

В файле labwork6.py представлен код функции которая обращается к API по url 'https://www.cbr-xml-daily.ru/daily_json.js' и возвращает словарь курсов валют для валют из списка currency_codes.

В файле labwork6(unittests).py представлены модульные тесты для функции `get_currencies()`. Тесты проверяют: корректность возвращаемого словаря валют; обработку сетевых ошибок и исключительных ситуаций; работу с пустыми и частично найденными списками валют; структуру и валидацию данных; граничные случаи и чувствительность к регистру; запись логов в поток вывода. Обеспечивается полное покрытие функциональности работы с внешним API.
