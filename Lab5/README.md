# В папке представлены результаты выполнения лабораторной работы номер 5.

В файле lab5.py представлен код для сравнительного анализа производительности различных методов вычисления факториала. Программа реализует четыре подхода к вычислению факториала: рекурсивный, итеративный, а также их мемоизированные версии.

В файле labwork5(unittests).py также представлен комплекс модульных тестов для проверки корректности реализации всех функций вычисления факториала. Тесты покрывают следующие аспекты: корректность вычислений для стандартных случаев (0, 1, 5, 10); идентичность результатов рекурсивного и итеративного подходов; проверка работы кэширования и соответствия мемоизированных версий оригиналам; независимость кэшей для разных функций и правильность кэширования; обработка особых ситуаций (ноль, отрицательные числа); работа с большими значениями и проверка структуры тестовых данных; соответствие результатов до и после применения мемоизации.
